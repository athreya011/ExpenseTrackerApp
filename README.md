# Expense Tracker App

An Android-based expense management application developed using Java and Firebase. The application allows users to track expenses, manage financial records, and perform real-time currency conversions through API integration.

## Features

- User Authentication using Firebase
- Expense Tracking and Management
- Real-Time Database Storage
- Currency Conversion using Exchange Rate API
- User-Friendly Mobile Interface
- Secure Cloud Data Storage

## Tech Stack

### Frontend
- Java
- XML
- Android Studio

### Backend
- Firebase Authentication
- Firebase Firestore

### APIs
- Currency Exchange API
